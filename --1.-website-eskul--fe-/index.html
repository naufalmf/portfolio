Divisi Networking

Melakukan konfigurasi jaringan melalui sistem operasi Linux yang diinstall pada VirtualBox dan dengan aplikasi Winbox kami melakukan konfigurasi MicroTik menggunakan MAC Address atau protokol IP.

    Kegiatan :

  • Pembuatan kabel LAN
  • Melakukan transfer data melalui jaringan
  • Membuat IP Address
  • Mengkonfigurasi jaringan
  • Mengkonfigurasi DNS Server
  • Membuat Domain

Divisi Programming

Melakukan pembuatan website menggunakan HTML(Hyper Text Markup Language), dan mengatur tampilan website menggunakan CSS(Cascading Style Sheets) juga dibantu dengan JS(JavaScript) yang bekerja pada Client-Side agar halaman website bisa interaktif.

Pengembangan website menggunakan bahasa pemrograman PHP(Hypertext Prepocessor) yang berjalan pada Server-Side, dan menggunakan database MySQL sebagai sarana penyimpanan data.

    Kegiatan :

  • Pembuatan website menggunakan HTML
  • Mengatur tampilan website menggunakan CSS
  • Pemahaman responsive pada CSS
  • Pemahaman bahasa pemrograman PHP
  • Menghubungkan website dengan database MySQL
  • Pemahaman bahasa pemrograman JavaScript
